# Music Player Web App

A simple and user-friendly Music Player built using Flask and JavaScript that allows users to search songs, listen to music, and download tracks easily. The interface is clean, responsive, and lightweight.

# Features
## Listen to Music

* Users can click any song to play the music
* Audio plays directly in the browser
* Lightweight and fast playback

## Search Music

* Search any song using the search bar
* Integrated with a Music API
* Shows search results instantly

## Download Music

* One-click song download
* Secure API-based downloading

## Responsive UI

* Clean and simple user interface
* Built with HTML, CSS, and JavaScript

## Backend (Flask + Python)

* Flask handles API requests
* Fetches music data
* Streams songs to the frontend

## Tech Stack

## Frontend
* HTML
* CSS
* JavaScript
* Backend
* Python
* Flask

## APIs

* Music Search API (custom or third-party)
